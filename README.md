# Modern Todo List Application 🚀

A sleek and efficient todo list application built with React, leveraging Redux, Tailwind CSS, and modern web development practices.

## ✨ Features

### 🔹 Task Management
- Intuitive task creation and management
- Inline task editing
- Drag-and-drop task reordering
- Easy task completion tracking

### 🔹 Smart Organization
- Flexible sorting (by name or date)
- Instant task search
- Dedicated completed tasks view
- Clean, user-friendly interface

### 🔹 Advanced Capabilities
- Persistent local storage
- Keyboard shortcut support
- Real-time search filtering
- Fully responsive design

## 🛠 Tech Stack

| Category | Technology |

## 🚀 Quick Start

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository
```bash
git clone https://github.com/yourusername/todo-app.git
cd todo-app
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```
